# MVPUp
Mvp架构逐步升级demo  
demo1:简单Mvp  
demo2:进阶版(泛型+抽象)  
demo3:终极版(泛型+高度抽象+双重代理)  

双重代理：第一重为生命周期代理，第二重为绑定和解绑的代理  

API使用的是曾经项目的，只是简单的获取到图片id
